# AI Job Application Tracker - Frontend

## Overview

Frontend for an AI-powered job application tracking platform.

Users can:
- Manage job applications
- Upload resumes
- Generate AI resume feedback
- Generate AI cover letters
- Perform skill gap analysis

## Tech Stack

- React
- Vite
- Tailwind CSS
- Axios
- React Router

## Features

### Authentication
- JWT Login
- JWT Registration
- Protected Routes

### Job Applications
- Create Application
- Update Application
- Delete Application
- Search & Filter

### Resume Management
- Upload Resume
- Delete Resume
- Select Resume for AI Features

### AI Features
- Resume Feedback
- Cover Letter Generation
- Skill Gap Analysis

```bash
npm install
